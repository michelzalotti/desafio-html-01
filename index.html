Utilizando Tags de Título

voltar

Podemos definir títulos de diversos níveis no HTML. Para isso existem as tags h1,h2,h3,h4,h5 e h6.

A tag h1 possui um 'peso' maior dentro da estrutura semântica de uma página sendo a principal dentre elas e a que possui uma formatação visual mais "chamativa", como tamanho de fonte maior, estilo de negrito mais acentuado e margens.

Devemos respeitar a hierarquia dessas tags para mantermos um código coerente e visualmente atraente.

Utilizando Parágrafos

voltar

É muito simples utilizar parágrafos no HTML, para isso existe a tag p.

Todo parágrafo deverá ser envolvido por essa tag, sendo que ela deve ser aberta no início e fechada no final de cada parágrafo.

Destacando Textos

voltar

Existem diversas formas para destacarmos textos no HTML. Podemos sublinhar palavras utilizando a tag u, colocar em negrito utilizandos as tags b ou strong e em itálico utilizando as tags i ou em, usar efeito de marcador de textos com a tag mark etc.

É possível definir um texto que foi definido como excluído, utilizando a tag del, mas que ainda permanece no original para não perder a informação de que aquele conteúdo foi substituído por outro utilizando a tag ins.

Podemos também definir um texto com uma formatação pequena usando a tag small, superescrito com a tag sup e subscrito com a tag sub.

Criando Links

voltar

Páginas de sites possuem ligações umas com as outras através de links, esses links são criados pela tag a. Um link também pode ser criado para navegar dentro do conteúdo da própria página ou para ir para outras páginas do mesmo site ou de outros sites.

Podemos definir como essas páginas irão abrir no navegador através do atributo target da tag, podendo abrir na mesma guia, em uma nova guia ou em uma nova janela do navegador.

Definindo Listas Ordenadas

voltar

Podemos definir listas ordenadas que seguem uma sequência numérica, alfabética ou algarismos romanos. Para criar-mos listas devemos definir a lista através da tag ol e cada item da lista deverá ser definida com a tag li.

Para definir se a lista será numérica, alfabética ou números romanos, podemos definir um atributo type para a tag com o valor 1, "A" ou "a","I" ou "i".

Definindo Listas Não Ordenadas

voltar

Também podemos definir listas não ordenadas. Essas listas não possuem marcadores sequenciais mas sim símbolos em cada item da lista.

Para definirmos uma lista usamos a tag ul e os itens da lista usamos a tag li.

  • Item 1
  • Item 2
  • Item 3
